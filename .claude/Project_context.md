### Project context
Food Image-to-Recipe Retrieval with Ingredient-Aware Fusion
1. Difficulty Level: Hard
2. Project Title: Food Image-to-Recipe Retrieval with Ingredient-Aware Fusion
3. Problem Statement: Food images rarely reveal all ingredients, while recipes contain structured ingredients and instructions. This project retrieves matching recipes from images using visual embeddings and ingredient text representations. It is medium difficulty when scoped to a subset, but students must evaluate both image-to-text and text-to-image retrieval.
4. Dataset along with URL: Recipe1M+ - http://pic2recipe.csail.mit.edu/
